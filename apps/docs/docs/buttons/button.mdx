# Button

Primary action button to trigger an operation.

```tsx
import { Button } from "rizzui/button";
```

<div className="mb-20 md:mb-28" />

## Default

The default style of **Button** component.

<Paper>
  <Button>Button</Button>
</Paper>

```tsx
import { Button } from "rizzui/button";

export default function App() {
  return <Button>Button</Button>;
}
```

## Variants

You can change the style of the **Button** using [**variant**](#button-variants) property.

<Paper>
  <div className="flex items-center justify-around gap-4 w-full flex-wrap">
    <Button variant="solid">Solid</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="flat">Flat</Button>
    <Button variant="text">Text</Button>
    <Button variant="danger">Danger</Button>
  </div>
</Paper>

```tsx
import { Button } from "rizzui/button";

export default function App() {
  return (
    <>
      <Button variant="solid">Solid</Button>
      <Button variant="outline">Outline</Button>
      <Button variant="flat">Flat</Button>
      <Button variant="text">Text</Button>
      <Button variant="danger">Danger</Button>
    </>
  );
}
```

## Loading

You can set the loading state of the **Button** component using **isLoading** property.

<Paper>
  <div className="flex items-center justify-around gap-4 w-full flex-wrap">
    <Button isLoading={true}>Solid</Button>
    <Button isLoading={true} variant="outline">
      Outline
    </Button>
    <Button isLoading={true} variant="flat">
      Flat
    </Button>
    <Button isLoading={true} variant="danger">
      Danger
    </Button>

import { Button } from "rizzui/button";

        Outline
      </Button>
      <Button isLoading={true} variant="flat">

      </Button>
      <Button isLoading={true} variant="danger">

## With Icon

You can set any Icon at any possition.

<Paper>
  <Button>
    <span>View Details</span>{" "}
    <ArrowRightIcon
      strokeWidth="2"
      className="h-4 w-4 ml-2"
    />
  </Button>
</Paper>

```tsx
import { Button } from "rizzui/button";
import { ArrowRightIcon } from "@heroicons/react/24/outline";

export default function App() {
  return (
    <Button>
      <span>View Details</span>{" "}
      <ArrowRightIcon
        strokeWidth="2"
        className="h-4 w-4 ml-2"
      />
    </Button>
  );
}
```

### Button Props

Here is the API documentation of the Button component. And the rest of the props are the same as the original html button. You can use props like **id**, **title**, **onClick**, **onFocus**, **onBlur** etc.

<div className="overflow-x-auto">

|   Props   |                    Type                    |                                       Description                                       |  Default   |
| :-------: | :----------------------------------------: | :-------------------------------------------------------------------------------------: | :--------: |
|    as     |             `button` or `span`             |                                        Render as                                        | `"button"` |
| children  |             `React.ReactNode`              |                           Accepts everything React can render                           |    \_\_    |
|   type    |        [ButtonTypes](#button-types)        |                          Set the original HTML type of button                           | `"button"` |
|  variant  |     [ButtonVariants](#button-variants)     |                                 Set the button variants                                 | `"solid"`  |
|   size    |        [ButtonSizes](#button-sizes)        | Set the size of the component. <br /> `"sm"` is equivalent to the dense button styling. |  `"md"`    |
| isLoading |                 `boolean`                  |                            Set the loading status of button                             |    \_\_    |
|  disabled |                 `boolean`                  |                              Disabled state of the button                               |    \_\_    |
|   loader  |             `React.ReactNode`              |                     Custom Loader component to show when isLoading                      |    \_\_    |
| className |                  `string`                  |                           Add custom classes for extra style                            |    \_\_    |
|    ref    |          `Ref<HTMLButtonElement>`          |                                       forwardRef                                        |    \_\_    |
|    ...    | `ButtonHTMLAttributes` or `HTMLAttributes` |              native props like **onClick**, **title**, **aria-label** ...               |    \_\_    |

</div>
