# Rate

A component for getting word of experience of the user.

```tsx
import { Rate } from 'rizzui';
```

<div className="mb-28" />

## Default

The default style of **Rate** component.

<Paper>
	<Rate className="[&_li]:!mt-0" />
</Paper>

```tsx
import { Rate } from 'rizzui';

export default function App() {
	return <Rate />;
}
```
