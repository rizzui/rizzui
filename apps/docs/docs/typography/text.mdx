# Text

Text component is the used to render text and paragraphs within an interface using well-defined typographic styles. It renders a **p** tag by default.

```jsx
import { Text } from "rizzui";
```

<div className="mb-20 md:mb-28" />

## Default

The default style of the **Text**.

<Paper>
  <Text>The brown fox jumps over the lazy dog</Text>
</Paper>

```jsx
import { Text } from "rizzui";

export default function App() {
  return <Text>The brown fox jumps over the lazy dog</Text>;
}
```

### Text Props

Here is the API documentation of the Text component.

<div className="overflow-x-auto">

|   Props    |              Type              |                               Description                                | Default  |
| :--------: | :----------------------------: | :----------------------------------------------------------------------: | :------: |
|     as     |     [TextTags](#text-tags)     |                                                                          |  `"p"`   |
|   title    |        `abbr` `string`         |                 title attribute only appear when tag is                  |   \_\_   |
| fontWeight | [TitleWeights](#title-weights) | Add font-weight, fontWeight='normal' won't work with b, strong and q tag | `"bold"` |
| className  |            `string`            |                    Add custom classes for extra style                    |   \_\_   |

</div>

<div className="mb-10" />

### Text Tags

```ts
type TextTags =
  | "p"
  | "b"
  | "span"
  | "strong"
  | "abbr"
  | "q"
  | "del"
  | "em"
  | "i"
  | "kbd"
  | "mark"
  | "small"
  | "sub"
  | "sup";
```

### Title Weights

```ts
type TitleWeights = "bold" | "normal" | "medium" | "semibold";
```
