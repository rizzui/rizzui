# AdvancedRadio

A basic widget for getting the user input of radio with advanced design.

```jsx
import { AdvancedRadio } from "rizzui";
```

### AdvancedRadio Props

Here is the API documentation of the AdvancedRadio component. And the rest of the props of AdvancedRadio are the same as the original html input field.

<div className="overflow-x-auto">

|      Props       |                  Type                   |                                   Description                                   | Default |
| :--------------: | :-------------------------------------: | :-----------------------------------------------------------------------------: | :-----: |
|     children     |               `ReactNode`               |                            Pass content as children                             |  \_\_   |
|     disabled     |                `boolean`                |                     Whether the checkbox is disabled or not                     |  \_\_   |
|       size       |   [RadioSizes](#advanced-radio-sizes)   | The size of the component.<br /> "sm" is equivalent to the dense input styling. | `"md"`  |
|     rounded      | [RadioRounded](#advanced-radio-rounded) |                            The rounded variants are:                            | `"md"`  |
|  inputClassName  |                `string`                 |                      Add inputClassName to style the input                      |  \_\_   |
| contentClassName |                `string`                 |                     Add contentClassName to style the input                     |  \_\_   |
|    className     |                `string`                 |                 Add custom classes to the root of the component                 |  \_\_   |
|       ref        |         `Ref<HTMLInputElement>`         |                                                                                 |  \_\_   |
|       ...        |          `InputHTMLAttributes`          |     native props like **value**, **onChange**, **onFocus**, **onBlur** ...      |  \_\_   |

</div>
