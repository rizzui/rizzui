# Announcement

A component for new product announcement or news.

```jsx
import { Announcement } from "rizzui";
```

<div className="mb-20 md:mb-28" />

## Default

The default style of **Announcement** component.

<Paper>
  <Announcement
    className="flex-wrap"
    badgeText="Trending"
    highlightedText="25% discount"
  >
    on our new product
  </Announcement>
</Paper>

```jsx
import { Announcement } from "rizzui";

export default function App() {
  return (
    <Announcement
      className="flex-wrap"
      badgeText="Trending"
      highlightedText="25% discount"
    >
      on our new product
    </Announcement>
  );
}
```

## With Icons

You can add icons to **Announcement** using **startIcon** & **endIcon** property.

<Paper className="gap-4 justify-start items-start">
  <Announcement
    className="flex-wrap"
    highlightedText="25% discount"
    color="primary"
    startIcon={<MegaphoneIcon className="w-5 text-primary-dark" />}
  >
    on our every product
  </Announcement>
  <Announcement
    className="flex-wrap"
    badgeText="Trending"
    highlightedText="25% discount"
    color="danger"
    endIcon={<ChevronRightIcon className="w-3.5 ml-1" />}
  >
    on our every product
  </Announcement>
</Paper>

```jsx
import { Announcement } from "rizzui";
import { ChevronRightIcon, MegaphoneIcon } from "@heroicons/react/24/outline";

export default function App() {
  return (
    <>
      <Announcement
        className="flex-wrap"
        highlightedText="25% discount"
        color="primary"
        startIcon={<MegaphoneIcon className="w-5 text-primary-dark" />}
      >
        on our every product
      </Announcement>
      <Announcement
        className="flex-wrap"
        badgeText="Trending"
        highlightedText="25% discount"
        color="danger"
        endIcon={<ChevronRightIcon className="w-3.5 ml-1" />}
      >
        on our every product
      </Announcement>
    </>
  );
}
```

### Announcement Props

Here is the API documentation of the Announcement component.

<div className="overflow-x-auto">

|          Props           |                     Type                     |                       Description                        |   Default   |
| :----------------------: | :------------------------------------------: | :------------------------------------------------------: | :---------: |
|        badgeText         |                   `string`                   |      Pass badge text to have badge at the beginning      |    \_ \_    |
|     highlightedText      |                   `string`                   |          Pass highlighted text at the beginning          |    \_ \_    |
|         children         |                                              |                                                          |    \_ \_    |
|           size           |   [AnnouncementSizes](#announcement-sizes)   |                The size of the component.                | `"DEFAULT"` |
|         rounded          | [AnnouncementRounded](#announcement-rounded) |                The rounded variants are:                 | `"DEFAULT"` |
|          color           |  [AnnouncementColors](#announcement-colors)  |                 Choose color variations                  | `"DEFAULT"` |
|         bgColor          |                `white` `gray`                |            Background color of the component             |  `"white"`  |
|        className         |                   `string`                   |     Pass className to customize announcement design      |    \_ \_    |
|        startIcon         |                 `ReactNode`                  |               Place icon at the beginning                |    \_ \_    |
|         endIcon          |                 `ReactNode`                  |                  Place icon at the end                   |    \_ \_    |
|      badgeClassName      |                   `string`                   |            Pass badgeClassName to style badge            |    \_ \_    |
| highlightedTextClassName |                   `string`                   | Pass hightlightedTextClassName to style highlighted text |    \_ \_    |

</div>
