# Table

A content area which can be collapsed and expanded. Completely unstyled, fully accessible UI components, designed to integrate beautifully with Tailwind CSS. Visit our [TanStack Table](/docs/integrations/tan-stack) integration page for a full table example with **TanStack Table**.

```tsx
import { Table } from "rizzui-v1";
```

<div className="mb-20 md:mb-28" />

## Minimal

You can change the style of **Table** by changing property [**variant**](#table-variants).

<Paper>
  <TableDemo variant="minimal" />
</Paper>

```tsx
import { Table } from "rizzui-v1";

export default function App() {
  return (
    <Table variant="minimal">
      <Table.Header>...</Table.Header>
      <Table.Body>...</Table.Body>
    </Table>
  );
}
```
