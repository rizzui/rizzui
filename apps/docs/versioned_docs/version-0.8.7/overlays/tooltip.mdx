# Tooltip

Tooltip displays informative text when users **hover** or **focus** an element.

```jsx
import { Tooltip } from "rizzui-v0";
```

<div className="mb-20 md:mb-28" />

## Default

The default style of the **Tooltip** component.

<Paper>
  <Tooltip content={"I'm your tooltip"}>
    <Button variant="outline">Hover Me!</Button>
  </Tooltip>
</Paper>

```tsx
import { Tooltip, Button } from "rizzui-v0";

export default function App() {
  return (
    <Tooltip content={"I'm your tooltip"}>
      <Button variant="outline">Hover Me!</Button>
    </Tooltip>
  );
}
```

## With Custom Content

You can use custom content inside **Tooltip**

<Paper>
  <Tooltip
    content={
      <div className="w-40 text-start">
        <div className="inline-flex items-center gap-2 text-base mb-1.5">
          <InformationCircleIcon className="w-5 h-5" />
          <Text fontWeight="medium">Tooltip Title</Text>
        </div>
        <Text className="text-sm text-gray-200">
          This is a tip to help you accomplish a task.
        </Text>
      </div>
    }
    placement="bottom-start"
  >
    <Button variant="outline">Custom Content</Button>
  </Tooltip>
</Paper>

```tsx
import { Tooltip, Text, Button } from "rizzui-v0";
import { InformationCircleIcon } from "@heroicons/react/24/solid";

function CustomContent() {
  return (
    <div className="w-40 text-start">
      <div className="inline-flex items-center gap-2 text-base mb-1.5">
        <InformationCircleIcon className="w-5 h-5" />
        <Text fontWeight="medium">Tooltip Title</Text>
      </div>
      <Text className="text-sm text-gray-600">
        This is a tip to help you accomplish a task.
      </Text>
    </div>
  );
}

export default function App() {
  return (
    <Tooltip content={<CustomContent />} placement="bottom-start">
      <Button variant="outline">Custom Content</Button>
    </Tooltip>
  );
}
```

### Tooltip Props

Here is the API documentation of the Tooltip component. You can use the following props to create a demo of tooltip.

<div className="overflow-x-auto">

|     Props      |                   Type                   |                   Description                   |   Default   |
| :------------: | :--------------------------------------: | :---------------------------------------------: | :---------: |
|    content     |               `ReactNode`                |               Content for tooltip               |    \_\_     |
|   placement    | [TooltipPlacements](#tooltip-placements) |        Supported Tooltip Placements are:        |  `bottom`   |
|    children    |   [TooltipChildren](#tooltip-children)   |      Pass children which will have tooltip      |    \_\_     |
|     color      |     [TooltipColors](#tooltip-colors)     |              Change Tooltip color               | `"primary"` |
|      size      |      [TooltipSizes](#tooltip-sizes)      |          Supported Tooltip sizes are:           |   `"md"`    |
|    rounded     |    [TooltipRounded](#tooltip-rounded)    |            The rounded variants are:            |   `"md"`    |
|     shadow     |    [TooltipShadows](#tooltip-shadows)    |         Supported tooltip shadows are:          |   `"md"`    |
|   className    |                 `string`                 |   Add custom classes to the tooltip container   |    \_\_     |
|   showArrow    |                `boolean`                 | Whether tooltip arrow should be shown or hidden |   `true`    |
| arrowClassName |                 `string`                 |     Add custom classes to the tooltip arrow     |    \_\_     |

</div>
