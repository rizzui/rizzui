# Button

Primary action button to trigger an operation.

```tsx
import { Button } from "rizzui-v0";
```

## Default

The default style of **Button** component.

<Paper>
  <Button>Button</Button>
</Paper>

```tsx
import { Button } from "rizzui-v0";

export default function App() {
  return <Button>Button</Button>;
}
```

<Paper>

    <Button variant="outline">Button</Button>

### Button Props

Here is the API documentation of the Button component. And the rest of the props are the same as the original html button. You can use props like **id**, **title**, **onClick**, **onFocus**, **onBlur** etc.

<div className="overflow-x-auto">

|   Props   |                    Type                    |                                       Description                                       |   Default   |
| :-------: | :----------------------------------------: | :-------------------------------------------------------------------------------------: | :---------: |
|    as     |             `button` or `span`             |                                        Render as                                        | `"button"`  |
| children  |             `React.ReactNode`              |                           Accepts everything React can render                           |    \_\_     |
|   type    |        [ButtonTypes](#button-types)        |                          Set the original HTML type of button                           | `"button"`  |
|  variant  |     [ButtonVariants](#button-variants)     |                                 Set the button variants                                 |  `"solid"`  |
|   size    |        [ButtonSizes](#button-sizes)        | Set the size of the component. <br /> `"sm"` is equivalent to the dense button styling. |   `"md"`    |
|  rounded  |      [ButtonRounded](#button-rounded)      |                        Set border radius of the button component                        |   `"md"`    |
|   color   |       [ButtonColors](#button-colors)       |                                   Change button color                                   | `"primary"` |
| isLoading |                 `boolean`                  |                            Set the loading status of button                             |    \_\_     |
| disabled  |                 `boolean`                  |                              Disabled state of the button                               |    \_\_     |
| className |                  `string`                  |                           Add custom classes for extra style                            |    \_\_     |
|    ref    |          `Ref<HTMLButtonElement>`          |                                       forwardRef                                        |    \_\_     |
|    ...    | `ButtonHTMLAttributes` or `HTMLAttributes` |              native props like **onClick**, **title**, **aria-label** ...               |    \_\_     |

</div>
